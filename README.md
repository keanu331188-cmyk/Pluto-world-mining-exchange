# Pluto World Mining Exchange

A polished broker website with a client-facing landing page, investment plans, and an admin dashboard for reviewing KYC documents and withdrawal requests.

## Features
- premium investment broker landing page
- plan showcase for investor tiers
- admin sign-in flow for compliance staff
- persistent KYC and withdrawal request management
- regression test for approval persistence

## Run locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   npm start
   ```
3. Open http://127.0.0.1:3000/
